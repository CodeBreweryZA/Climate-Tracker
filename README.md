## Features
- Climate data visualization
- Educational content on climate change

## How to Run
Open `index.html` in any modern web browser. No server required.

```bash
# Clone the repo
git clone https://github.com/CodeBreweryZA/Climate-Tracker.git

# Open directly in browser
cd Climate-Tracker
start index.html
```

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Chart.js for data visualization
